# Copy AI-ready context

Use when the user wants to paste structured workspace memory into a new AI chat.

1. Ensure the Contorium VS Code extension is installed and a folder workspace is open.
2. Run command palette action **Contorium: Copy AI-ready context (clipboard)** (`contora.exportAIContext`).
3. If **Current focus** is empty, remind the user that setting a focus improves direction (non-blocking).
4. Paste the clipboard content into the target AI chat and continue from the exported TASK / WORKSPACE FOCUS / ACTIVE FILES sections.
