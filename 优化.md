## ⚙️ TypeScript 实现

```ts
function detectDrift(ctx) {
  const currentVector = buildCurrentVector(ctx.pil, ctx.cil, ctx.timeline);
  const goalVector = buildGoalVector(ctx.pik);

  const similarity = cosineSimilarity(currentVector, goalVector);

  const driftScore = 1 - similarity;

  return {
    driftScore,
    severity: classifySeverity(driftScore),
    driftType: classifyDrift(ctx),
    explanation: generateDriftExplanation(ctx)
  };
}
```

function classifySeverity(score) {
